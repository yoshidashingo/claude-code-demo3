# ユビキタス言語定義

## ドメイン用語

| 日本語 | 英語 | コード上の命名 | 定義 |
|--------|------|---------------|------|
| タスク | Task | `Task` | ユーザーが管理する作業項目 |
| タスクタイトル | Task Title | `title` | タスクの名称・説明 |
| 完了状態 | Completed | `completed` | タスクが完了したかどうかの状態 |
| 未完了 | Active | `active` | まだ完了していないタスク |
| フィルター | Filter | `FilterType` | タスクの表示条件 |

## ビジネス用語

| 用語 | 定義 |
|------|------|
| タスク追加 | 新しいタスクを作成してリストに追加すること |
| タスク完了 | タスクの完了状態をtrueに切り替えること |
| タスク削除 | タスクをリストから永久に除去すること |
| タスク編集 | 既存タスクのタイトルを変更すること |
| フィルタリング | 条件に基づいてタスクの表示を絞り込むこと |

## UI/UX用語

| 用語 | 定義 |
|------|------|
| タスク入力フォーム | 新規タスクのタイトルを入力するテキストフィールド |
| フィルターボタン | フィルター条件を切り替えるボタン群 |
| タスクリスト | タスクの一覧を表示する領域 |
| タスクアイテム | 個別のタスクを表示する行 |
| フッター | 残りタスク数を表示する領域 |
| インライン編集 | タスクリスト上で直接タイトルを編集する操作 |

## フィルタータイプ

| 値 | 日本語 | 説明 |
|----|--------|------|
| `all` | すべて | 全タスクを表示 |
| `active` | 未完了 | 未完了のタスクのみ表示 |
| `completed` | 完了済み | 完了済みのタスクのみ表示 |
