# 初回実装 - 要求内容

## 概要

食品・グルメ専門ECサイト「おいしいマルシェ」のMVP初回実装を行う。
フロントエンドのみのReact SPAとして構築し、モックデータを使用する。

## 実装する機能

### 1. 商品一覧表示
- カード形式で商品を表示（画像、商品名、価格、カテゴリ、お気に入りボタン）
- カテゴリによるフィルタリング（全て / 和菓子 / 洋菓子 / 調味料 / 飲料 / 加工食品）
- レスポンシブグリッド表示（モバイル1列、タブレット2列、デスクトップ3〜4列）

### 2. 商品詳細表示
- 商品画像の大きな表示
- 商品情報（名前、価格、説明、カテゴリ、在庫状況）
- 「カートに追加」ボタン（在庫なし時は非活性）
- 「お気に入りに追加/解除」ボタン
- 商品一覧への戻るリンク

### 3. カート機能
- 商品のカート追加（同一商品は数量加算）
- カート内商品の数量変更（増減ボタン）
- カート内商品の個別削除
- 小計・合計金額の表示
- カート空時のメッセージ表示
- ヘッダーにカート内商品数のバッジ表示

### 4. お気に入り機能
- 商品のお気に入り登録/解除（トグル操作）
- お気に入り一覧ページでの表示
- 商品カード・商品詳細からの操作
- お気に入り空時のメッセージ表示
- ヘッダーにお気に入り件数のバッジ表示

## 受け入れ条件

- [ ] 4画面（商品一覧、商品詳細、カート、お気に入り）が正常に表示される
- [ ] 画面間の遷移が正常に動作する
- [ ] カート操作（追加・削除・数量変更・合計計算）が正しく動作する
- [ ] お気に入り操作（追加・解除・一覧表示）が正しく動作する
- [ ] レスポンシブデザインが3ブレークポイントで正しく表示される
- [ ] TypeScript型チェックエラーがゼロ
- [ ] ESLintエラーがゼロ
- [ ] ビルドが成功する

## 制約事項

- フロントエンドのみで完結する（バックエンドAPIなし）
- 商品データはモックデータとして `src/data/products.ts` に定義
- 決済機能・ユーザー認証は含まない
- localStorage等のデータ永続化は含まない
- 技術スタック: React 18 + TypeScript 5 + Vite 5 + Tailwind CSS 3 + React Router 6
