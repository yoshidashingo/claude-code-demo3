# 初回実装 - 要求内容

## 概要
タスクマネージャーアプリケーションの初回実装。全基本機能を実装する。

## 実装する機能
1. タスク追加（テキスト入力 + Enter/ボタン）
2. タスク完了/未完了の切り替え
3. タスク削除
4. タスクのインライン編集
5. フィルタリング（すべて/未完了/完了済み）
6. 残りタスク数表示
7. localStorage によるデータ永続化

## 受け入れ条件
- すべての基本機能が動作すること
- Tailwind CSS でスタイリングされていること
- TypeScript の型エラーがないこと
- ESLint エラーがないこと
- レスポンシブデザインであること
