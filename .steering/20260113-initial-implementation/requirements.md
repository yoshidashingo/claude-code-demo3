# 初回実装 - 要求内容

## 概要
食品・グルメECサイトのMVP（最小限実装）を行う。

## 変更・追加する機能

### 1. 商品一覧機能
- 商品をカード形式で一覧表示
- カテゴリでフィルタリング
- 商品名、価格、画像を表示

### 2. 商品詳細機能
- 商品の詳細情報を表示
- カートへの追加
- お気に入り登録

### 3. カート機能
- 商品の追加・削除
- 数量変更
- 合計金額表示

### 4. お気に入り機能
- 商品のお気に入り登録・解除
- お気に入り一覧表示

## 受け入れ条件

- [ ] 商品一覧が正しく表示される
- [ ] 商品詳細ページに遷移できる
- [ ] カートに商品を追加・削除できる
- [ ] 数量を変更すると合計金額が更新される
- [ ] お気に入りに追加・削除できる
- [ ] レスポンシブデザインで表示される
- [ ] TypeScriptの型エラーがない
- [ ] ESLintエラーがない

## 制約事項
- フロントエンドのみの実装（バックエンドなし）
- モックデータを使用
- ローカルストレージでの永続化はMVP後
