# タスクマネージャー

シンプルで直感的なタスク管理アプリケーション。

## 機能

- タスクの追加・完了・削除・編集
- フィルタリング（すべて/未完了/完了済み）
- 残りタスク数の表示
- localStorage によるデータ永続化
- レスポンシブデザイン

## 技術スタック

- React 19 + TypeScript
- Vite
- Tailwind CSS 4

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```
